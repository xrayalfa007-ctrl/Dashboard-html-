# AquaTrack 2.0 Lite Dashboard A simple, mobile-friendly dashboard for monitoring solar-powered boreholes in real-time. ## Features - **Water Flow**: Live LPM reading - **Water Level**: Meters - **Battery**: Solar battery percentage - **System Status**: Operational/Offline ## Built for Mandero Water Solutions - RUWASSA Solar Borehole Project, Rivers State, Nigeria Node: AQUA_SOLAR_001 ## Tech Pure HTML + CSS. No database needed. Easy for community staff to use.
